position: static

1、盒子是普通的盒子，布局是常规流

2、top、left、bottom、right以及z-index属性被忽略

position: relative

1、position in normal flow(常规流中的定位)。定位的基准是该元素在常规流中本应处在的位置。

2、运用相对定位的元素没有脱离常规流（out of flow）， 且不会影响其他同级元素的布局

什么是常规流（normal flow）？

Boxes in the normal flow belong to a formatting context, which may be block or inline, but not both simultaneously. Block-level boxes participate in a block formatting context. Inline-level boxes participate in an inline formatting context.

在常规流中，盒一个接着一个排列。在块级格式化上下文(block formatting context)里面， 它们竖着排列；在行内格式化上下文(inline formatting context)里面， 它们横着排列。

创建Block Formatting Context的条件

  • 根元素
  • 浮动（float不为none）
  • 绝对定位元素（position为absolute或fixed）
  • 行内块（inline-block）、表格单元格（table-cell）、表格标题（table-caption）
  • overflow不为visible的元素
  • 弹性盒子（flex或inline-flex）

BFC 特性之一

在一个BFC中，盒子一个接一个，竖直的，从上到下排列。两个相邻的盒子的间距由margin控制。作为BFC的父元素不会和其子元素发生外边距坍塌（margin collapse）
